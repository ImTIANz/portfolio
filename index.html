Instructions

Sudoku is a number puzzle game where you need to fill a 9x9 grid with digits so that each column, each row, and each of the nine 3x3 sub grids that compose the grid contain all of the digits from 1 to 9.

How to Play:

  • Click on a number from the "Digits" section to select it.
  • Click on an empty tile in the grid to place the selected number.
  • If you make a mistake, the "Errors" count will increase.
  • Your goal is to complete the puzzle with as few errors as possible.
  • You can also record your solving time, just click the start button and the timer will start. If you want to reset the time, just click the reset button.
  • Once you're done, click the check button, to check if you're answer is correct.
